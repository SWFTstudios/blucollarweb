# Blu Collar Web Solutions

Website for Blu Collar Web Solutions - a web design and digital marketing
agency serving blue-collar businesses and contractors.

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Deployed on Vercel / Netlify
- Originally built in Webflow, rebuilt as a static site

## Local Development
No build step required. Open `index.html` directly in a browser,
or use a local server:

```bash
# Using VS Code Live Server extension (recommended)
# or
npx serve .
```

## Deployment
Push to `main` branch - auto-deploys via Vercel / Netlify.
